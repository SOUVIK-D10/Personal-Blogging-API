Article API

Interactive REST API Documentation

Endpoints

Overview

The Article API is a RESTful backend built using Spring Boot that allows managing articles with search, filter, and CRUD operations.

Base URL

/article_api
GET /search

Search Articles

Search articles by keyword, title, author, or date range.

GET /article_api/search?keyword=spring&filter=title
Optional Body (Date Filter)
{
  "startDate": "2024-01-01",
  "endDate": "2024-01-31"
}
GET /article

Get Articles

Fetch all articles or a single article by ID.

GET /article_api/article
GET /article_api/article?id=5
POST /article

Create Articles (Bulk)

[
  {
    "title": "Java Basics",
    "author": "Alice",
    "content": "Intro to Java",
    "tags": "java,backend"
  }
]
PUT /article

Update Article

{
  "id": 3,
  "title": "Updated Title",
  "author": "Bob",
  "content": "Updated content",
  "tags": "java,update"
}
DELETE /article

Delete Article

DELETE /article_api/article?id=3
